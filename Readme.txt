
DDay.iCal is an iCalendar class library (also known as RFC2445) for .NET 2.0
and above. It's aimed at providing full RFC 2445 compliance, while providing
the highest level of compatibility with popular calendaring applications.

===============================================================

To get started developing with DDay.iCal, simply download the latest binary
release, add a reference to DDay.iCal.dll within your project, and you can
begin using it.  For examples, see the example project(s) included with the
binary distribution.

===============================================================

If you'd like to contribute to DDay.iCal, please let me know.  To compile
the project from source, you will need to download and install ANTLR
version 2.7.6 from antlr.org, and add the file antlr-2.7.6.exe to your
PATH environment variable.
